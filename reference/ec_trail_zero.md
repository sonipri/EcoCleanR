# Trail Zeros from the Coordinate Values

Trail Zeros from the Coordinate Values

## Usage

``` r
ec_trail_zero(coord)
```

## Arguments

- coord:

  A coordinate value in the numeric format of decimal degree

## Value

A numerical trailed coordinate value.

## Examples

``` r
ec_trail_zero(12.7000000)
#> [1] "12.7"
ec_trail_zero(45.000000)
#> [1] "45"
```
